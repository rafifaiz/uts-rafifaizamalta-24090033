## âœ¨ Deskripsi Singkat Project

Project ini merupakan implementasi dasar dari sistem manajemen data produk berbasis web (Client-Side) yang dibangun menggunakan HTML, CSS (desain modern dan responsif), dan JavaScript (untuk fungsionalitas dan manipulasi DOM).

### Fungsi Setiap Halaman

| Halaman | File | Fungsi Utama |
| :--- | :--- | :--- |
| **Halaman Login** | `index.html` | Mengautentikasi pengguna secara simulasi. <br> **Fungsionalitas JS:** Melakukan validasi *form* (cek *input* kosong) dan melakukan *redirect* ke Dashboard setelah login berhasil. |
| **Halaman Dashboard** | `dashboard.html` | Menampilkan ringkasan data penting (Total Produk, Total Penjualan, Pendapatan). <br> **Fungsionalitas JS:** Mengambil data *dummy* dan merender ke dalam kartu (*card summary*). Menyediakan navigasi cepat ke halaman List Data Produk. |
| **List Data Produk** | `products.html` | Menampilkan daftar produk dalam format tabel. <br> **Fungsionalitas JS:** Merender data produk ke tabel, menyediakan **tombol Aksi (Edit/Hapus)**. Fitur Hapus memungkinkan penghapusan data secara interaktif dari tabel (manipulasi DOM) dan array data. |
